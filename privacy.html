Privacy

Last updated: July 13, 2026

Short version

This is a static personal website. It does not use advertising trackers, analytics scripts, newsletter forms, or server-side guestbook submissions.

What is collected

The site itself does not intentionally collect personal data from visitors.

Some interactive features store data locally in your browser using localStorage:

  • language preference;
  • visual and accessibility settings;
  • 90s-mode guestbook entries;
  • retro visitor counter state;
  • terminal easter-egg command history and local virtual-filesystem state.

This local data stays in your browser and is not sent to this site’s server. You can remove it by clearing site data for sergey.works in your browser.

Hosting

The site is hosted on GitHub Pages. GitHub may process basic request logs needed to provide and secure the hosting service.

External services

Links to GitHub, LinkedIn, X, Bluesky, project websites, and other external sites open those services directly. Their privacy practices are controlled by those services.

The CV page loads web fonts from Google Fonts. If you open the CV page, your browser may request font files from Google.

Cookies

This site does not set tracking cookies. Because it does not use analytics or advertising cookies, it does not show a cookie consent banner.

Requests

If you have a privacy question or want to request removal of something that identifies you on this site, contact hello@sergey.works.

Changes

This notice may be updated when the site changes. The date above shows the latest update.
